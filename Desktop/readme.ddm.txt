# FilterShortStrings

Программа формирует массив строк длиной меньше либо равной 3 символам.

## Описание
- Ввод данных задан в коде (в виде массива).
- Программа без использования коллекций, только массивы.
- Результат — новый массив из строк, длина которых <= 3.

## Алгоритм
1. Подсчёт строк нужной длины
2. Создание нового массива
3. Копирование подходящих строк
4. Вывод результата

См. блок-схему в `algorithm.png`.
